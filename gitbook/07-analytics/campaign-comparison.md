# 캠페인 간 비교

여러 캠페인을 동시에 운영할 때, 어떤 캠페인이 효과적이고 어떤 캠페인이 개선이 필요한지 한눈에 비교합니다.

## 진입 방법

대시보드 좌측 메뉴 → **Analytics** → **Campaign Comparison**

**[사진첨부]** 캠페인 비교 페이지 진입

## 캠페인 선택

{% stepper %}
{% step %}
### 비교할 캠페인을 선택한다

드롭다운에서 2~5개의 캠페인을 선택합니다.

**[사진첨부]** 캠페인 멀티 셀렉트
{% endstep %}

{% step %}
### 기간을 맞춘다

기간을 통일하지 않으면 비교가 왜곡될 수 있습니다. 동일 기간(예: 최근 30일)을 지정합니다.

**[사진첨부]** 기간 셀렉터
{% endstep %}
{% endstepper %}

## 비교 뷰

**[사진첨부]** 캠페인 비교 테이블

| 지표 | 설명 |
|------|-----|
| Total Sessions | 기간 내 세션 수 |
| Completion Rate | 완료 비율 |
| Avg. Messages | 세션당 평균 메시지 수 |
| Avg. CSAT | 평균 만족도 |
| Top Purpose | 가장 많은 상담 목적 |
| BANT Coverage | BANT 네 항목 평균 파악률 |

숫자가 **녹색**이면 선택한 캠페인 중 최고, **빨간색**이면 최저입니다.

## 차트 비교

각 지표를 그룹 막대 차트로 비교합니다.

**[사진첨부]** 지표별 그룹 막대 차트

### 해석 예시

| 패턴 | 의미 | 대응 |
|------|-----|------|
| Completion Rate가 낮은 캠페인 | 고객이 대화를 중단하는 비율 높음 | 에이전트 프롬프트의 질문 밀도 점검 |
| Avg. Messages가 지나치게 많음 | 대화가 길어짐, 효율 저하 | 에이전트의 follow-up 로직 조정 |
| CSAT 격차 | 특정 에이전트나 캠페인 주제의 만족도 문제 | 해당 대화 로그 샘플링하여 품질 분석 |
| BANT Coverage 편차 | 특정 항목 수집이 부족 | 프롬프트에 해당 항목 강조 지시 추가 |

## 인사이트 활용

### 성공 요소 공유

성과가 우수한 캠페인의 에이전트 프롬프트를 다른 캠페인에 이식하여 공통 개선을 유도합니다.

### 실패 패턴 차단

Completion Rate가 30% 미만이거나 CSAT 3점 이하인 캠페인은 **임시 비활성화**하고 개선 작업을 진행합니다.

## 데이터 내보내기

비교 결과를 CSV로 내보내 경영진 보고나 정기 리뷰에 활용합니다.

```
campaign_name, sessions, completion_rate, avg_messages, avg_csat, top_purpose
FY26 Enterprise, 120, 72%, 18.5, 4.3, New Adoption
FY26 Migration,  87, 65%, 22.1, 4.0, Migration
FY26 SMB,        42, 54%, 14.3, 3.8, Cost Optimization
```
