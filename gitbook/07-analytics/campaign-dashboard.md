# 캠페인 대시보드

캠페인의 전체 세션을 집계해 성과와 패턴을 보여줍니다. 마케팅 효과 검토, 고객 관심사 파악, 후속 영업 우선순위 결정에 활용합니다.

## 진입 방법

캠페인 리스트에서 해당 캠페인 클릭 → **분석** 탭 선택

![캠페인 상세 분석 탭](../.gitbook/assets/07-campaign-dashboard-01-detail-tabs.png)

## 지표 카드

화면 상단에 핵심 지표 4개가 카드로 표시됩니다.

| 카드 | 보여주는 것 | 활용 |
|------|-----------|------|
| **전체 세션** | 생성된 세션 수 (활성/완료 구분 표시) | 캠페인 도달 규모 파악 |
| **완료율** | 완료 세션 / 전체 세션 (%) | 80% 이상이면 양호, 60% 미만이면 이탈 원인 점검 |
| **평균 대화 시간** | 완료된 세션의 평균 소요 시간 | 에이전트 프롬프트 효율성 판단 |
| **참여 기업** | 고유 기업 수와 최다 참여 기업 | 타겟 기업 도달 여부 확인 |

## 세션 추이 차트

날짜별 완료 세션 수를 바 차트로 보여줍니다. 캠페인 시작 후 접수가 언제 몰리는지, 특정 이벤트(세미나, eDM 발송) 이후 반응이 있는지 확인할 수 있습니다.

## 상담 목적 분포

고객이 대화 초반에 선택한 상담 목적을 차트로 집계합니다. 어떤 주제에 관심이 집중되는지 한눈에 파악됩니다.

## 참여 기업 테이블

세션에 참여한 기업을 세션 수 기준으로 정렬합니다. 같은 기업에서 여러 세션이 생성됐다면 관심도가 높다는 신호입니다.

## CSAT 피드백 테이블

대화 종료 후 고객이 남긴 피드백을 테이블로 보여줍니다.

| 컬럼 | 내용 |
|------|------|
| 고객 | 이름과 회사 |
| 평점 | 1~5점 (4점 이상 초록, 3점 파랑, 2점 이하 빨강) |
| 코멘트 | 고객이 남긴 자유 의견 |
| 완료일 | 세션 종료 시점 |

평점이 낮은 세션은 대화 로그를 확인하고 에이전트 프롬프트를 개선하는 데 참고합니다.

## 리포트 내보내기

캠페인 상세 페이지에서 **Export** 버튼을 클릭하면 현재 분석 데이터를 CSV로 다운로드합니다. 사내 보고서나 스프레드시트 분석에 활용합니다.
